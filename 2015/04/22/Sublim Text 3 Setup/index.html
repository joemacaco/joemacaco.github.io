Sublime Text 3 Setup

2015年4月22日
標籤
  • Sublime

以下介紹一些基礎設定，跟一些好用的插件介紹

安裝流程

下載 Sublime 3
安裝記的勾選 Add to explorer context menu 才能使用右鍵開啟選項

安裝完成後就安裝 Package Control 使用 Ctrl+` 開啟 Sublime控制台 輸入安裝指令送出

安裝完成後重開 Sublime 使用Ctrl+Shift+P開啟 Package Control 輸入Install

環境設定

開啟檔案目錄：File -> Open Folder
顯示目前開啟的檔案：View -> Side Bar -> Show Open Files
修改Sublime樣式：Preferences -> Color Scheme -> Colorsublime-Themes
修改字體大小：Preferences -> Settings-User -> "font_size": 14
取消自動偵測 Tab Size
修改設定：Preferences -> Settings
Settings-Default 則是預設的設定不可直接修改，可以看裡面有什麼設定，需要透過user的設定去修改，如下

安裝插件

其它插件：Colorsublime、jQuery、JsFormat、Sass、SCSS、SublimeTmpl、SFTP

Colorsublime

更改Sublime的樣式，安裝完成後在安裝版型

Emmet

在html時如果輸入input-pw + Tab就會自動轉換成<input class="form-control" type="password">

jQuery

jQuery指令加上Tab自動生成程式碼

JsFormat

Ctrl+Alt+F 就會自動排版

SublimeTmpl

File->New File(SublimeTmpl)　開啟新檔案自動填入網頁標籤

SFTP

資料夾右鍵 -> SFTP/FTP -> Map to Remote

SublimeLinter & SublimeLinter-php

提示使用者編寫的程式碼中存在的錯誤的寫法

ConvertToUTF8

檔案編碼轉換

Seti_UI

目前所使用的UI

phpfmt

自動轉換php符合 psr1 或 psr2 編寫格式

Markdown Preview

可以預覽Markdown或是轉換成HTML檔案

InputHelper

解決在Linux無法輸入中文問題

DocBlockr

方便使用註解
