MAC 安裝 Laravel 開發環境設定

2017年11月20日
標籤

  • PHP

記錄一下在mac上安裝開發環境的過程

HomeBrew

安裝HomeBrew

在終端機命令列提示貼上這個。

如果已經安裝過可透過 brew update 更新

Composer

安裝Composer

輸入 brew install composer 安裝
輸入 composer --version 查看安裝版本

PHP

安裝PHP

在Mac上安裝完PHP後，可以透過 php -v檢查版本，如果還是原本Mac內建版本，可以參考 brew info php71 裡面提供的資訊修正

PHP版本切換

當要安裝其它版本時會出現下面警示，透過 brew unlink php70 取消關連就可以順利執行這次安裝

透過安装php-version來切換版本

安裝完成後可以透過指令來切換版本

Laravel Valet

安裝Valet

若有發生valet安裝失敗
輸入 export PATH=~/.composer/vendor/bin:$PATH
執行 source ~/.bash_profile

使用Valet

詳細的使用可參考 Valet

  • park
    切換到你所建立的專案目錄後輸入 valet park
    接著建立laravel目錄 laravel new blog
    在瀏覽器輸入 blog.dev 即可瀏覽

Laravel

安裝Laravel工具

使用Composer下載Laravel

使用Laravel

建立新專案

MySQL

安裝MySQL

預設登入是沒有密碼，可以透過 mysql_secure_installation 來設置

phpMyAdmin

安裝phpMyAdmin

前往phpMyAdmin官網下載安裝包
解壓縮到Valet開發目錄底下
在瀏覽器輸入 phpmyadmin.dev 即可瀏覽
