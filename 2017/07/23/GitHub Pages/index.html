GitHub Pages 靜態網站

2017年7月23日
標籤
  • Github
  • Hexo

為了架設個人Blog所以才關注到，Github有提供將個人projects直接轉換靜態網站的服務，一些簡單的介紹可以觀看官方介紹影片What is GitHub Pages?

GitHub Pages官網底下有推薦使用Jekyll去建構靜態網頁，不過是使用Ruby去操作的，所以最後選擇使用Hexo當這次的介紹，Hexo官網也有繁體中文的文件所以建構起來也比較容易
